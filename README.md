# 🏠 House Price Prediction

A Machine Learning project that predicts house prices using regression algorithms.

## 📌 Project Objective

The goal of this project is to predict house prices based on different features like:

- Area
- Number of Bedrooms
- Bathrooms
- Floors
- Parking
- Furnishing Status

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🚧 Status

Project is currently under development.
